Esto es una prueba para control de versiones GIT

  • 0) antes de nada tenemos que configurar nuestros datos: - git config user.name "Yolanda Jimenez" - git config user.email "nukitas@gmail.com" * se supone que es con global (git global user.name...) pero da error
  • 1) creamos carpeta en local (yo creo rep_yolanda)
  • 2) generamos el repositorio. Para ello entramos a la carpeta desde git bash y ejecutamos: git init Si hacemos ls vemos que tenemos una carpeta .git creada en el directorio
  • 3) creamos en esta carpeta local(en rep_yolanda) el archivo en cuestión, que en mi caso soy yo (index.html).
  • 4) añadimos al repositorio local con "git add index.html" desde el mismo directorio en git bash
  • 5) Si ahora hacemos git status nos indica el archivo que está preparado para actualizarse.
  • 6) ahora hacemos el commit git commit -m "Creadala pagina de inicio"
  • 5) Si ahora hacemos git status nos indica que no hay nada. Si cambiamos el archivo index y hacemos status nos dice que existen cambios. ¿Qué hago para añadir estos cambios al repositorio?
  • 1) git add index.html (igual que antes)
  • 2) git commit -m "Modifico explicaciones"
